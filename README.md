## Lab 1 — OCI VM Deployment

Architecture Components:

- Oracle Cloud Infrastructure
- Virtual Cloud Network (VCN)
- Public Subnet
- Compute Instance (Oracle Linux)
- SSH Access
- Java Runtime Environment

Objectives:

1. Deploy a VM instance in OCI  
2. Configure networking and security rules  
3. Connect using SSH  
4. Install Java runtime  
5. Prepare environment for Java cloud applications
